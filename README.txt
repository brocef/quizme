To use the application, create a file in the cwd of the java executable called 'fieldtable.txt'.
The format of the file must follow this structure:

[# Of Fields]
List,Of,Field,Names
Field,Values,Go,Here
And,Here,And,here
Etc,Etc,Etc,Etc



A sample fieldtable has been included for your reference.
